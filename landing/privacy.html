Privacy Policy

Last updated: March 20, 2026 · Applies to: Claude Agent MCP Bridge (Chrome Extension)

This extension does not collect, transmit, or sell any personal data. All data remains on your local machine.

1. What This Extension Does

Claude Agent MCP Bridge is a Chrome extension that acts as a companion to the chrome-agent-mcp MCP server. It enables AI agents (such as Claude) to interact with your browser — navigating pages, clicking elements, filling forms, and inspecting DevTools — by relaying commands from the local MCP server to the browser.

2. Data Accessed

To function, the extension accesses the following browser data:

  • Tab information — URLs, titles, and favicons of open tabs, used to display the live session dashboard.
  • Tab groups — Creation and management of Chrome tab groups to isolate agent sessions.
  • Page content — The extension injects scripts into web pages on behalf of AI agent tool calls (e.g., to click a button or read page structure). This is done only when an AI agent actively requests it.
  • Local storage — Session state (group assignments, session IDs) is persisted locally via chrome.storage.local.

3. Data Retention and Transmission

No data accessed by this extension is ever transmitted to external servers, third parties, or the extension developer. All communication happens exclusively between:

  • Your local MCP server process (running on your machine), and
  • Your local Chrome instance.

Session state stored via chrome.storage.local stays on your device and is cleared when you uninstall the extension.

4. Permissions Used

  • tabs — Read tab URLs and titles for the session dashboard.
  • tabGroups — Create and manage tab groups per agent session.
  • scripting — Inject scripts into pages as instructed by the local AI agent.
  • storage — Persist session state locally.
  • host_permissions (<all_urls>) — Required so the extension can operate on any tab the AI agent is directed to.

5. Who Has Access to Your Data

Only you. This extension operates entirely locally. The developer of this extension has no access to your browsing data, tab content, or any other information.

6. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect information from children.

7. Changes to This Policy

If this policy changes, the updated version will be published at this URL and the "Last updated" date above will be revised.
